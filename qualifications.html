  • CompTIA A+

    • This certification is a general knowledge certification that gives me the knowledge and skills to work in any Technical Support position. Additionally, this certification give enough of a baseline knowledge of computer systems, that it is great asset in any IT position at any level. Finally, this certification compounds with others to almost exponentially increase my skills in IT.

  • CompTIA Server+

    • Often this certification is known as A+ but for servers. This fine tunes the knowledge gained by A+ to increase knowledge and skills in server administration and maintenance. Server+ is a great asset in any position, as knowledge of servers and clients is mandatory for support positions. This is particularly true in a world where cloud technologies and hosted services are growing.

  • CompTIA Network+

    • Network+ is the foundational certification for understanding computer networks. It is essentially the general version of the CCNA certification. Network+ is very useful to have in any position, as it is a field that is inherently part of every single aspect of IT. What use is any computer system to a global company if it is unable to connect to anything else? Network+ increases my value as a programmer especially, as it teaches a discipline that many programmers do not invest in.

  • LPI Linux Essentials

    • This certification increases the range of computer systems that I have fluency in. While Linux is different from MacOS, familiarity with the Terminal allows for expertise to be exercised on any Unix-based systems. This certification teaches the fundamental concepts and means by which an IT professional can administrate, network, and troubleshoot any Linux system.







  • MTA Server Adminstration - MTA Security Fundamentals - MTA Networking Fundamentals

    • MTA Security Fundamentals, Networking, and Server Adminstration are part of a suite of certifications that are no longer offered, however, the knowledge gained is still very much valid. MTA Networking and MTA Security hone my knowledge of cybersecurity and networking, while MTA Server Adminstration is a certification known for its focus on Active Directory. This knowledge of AD makes me very useful as a system administrator or technical support specialist.
